Como Jogar

Seu objetivo é adivinhar o nome do filme a partir de uma sinopse criativa e, muitas vezes, bem estranha!

  • Você tem 3 chances para acertar cada filme.
  • Um palpite muito próximo da resposta correta te dará um feedback de "Quase lá!", mas ainda consome uma chance.
  • A partir do segundo filme, você pode clicar em "Nova descrição" para obter uma sinopse diferente para o mesmo filme, ao custo de uma chance.
  • Use o botão "Pular" se não souber a resposta para passar para o próximo desafio.

O sistema ignora maiúsculas/minúsculas, acentos e a maioria dos símbolos na sua resposta. Divirta-se!
